DLVid. Download videos from websites.
(c) 2014 by Michael Price, MIT License

Usage:
  dlvid download <url>... [--prefix=<prefix>] [--output=<file>] [--directory=<directory>]
  dlvid info <url> ...
  dlvid -h | --help
  dlvid --version

Options:
  -h --help                  Show this screen.
  -v --version               Show the current version.
  -p --prefix=<prefix>       Prefix output files [default: dlvid].
  -o --output=<file>         The destination the file should be sent to if it's a single file [default: vid].
  -d --directory=<directory> The directory the files should be placed in if mulitple files are being retreived  [default: ./].
